# About Brazilian Delight
**Brazilian Delight** is an addon for the [Farmer's Delight](https://modrinth.com/mod/farmers-delight) mod. It adds recipes, crops and features related to the brazilian culture!

**[Brazil](https://en.wikipedia.org/wiki/Brazil)** is a tropical country with a rich and diverse culture, and this mod aims to bring some of that to the game. There are many things to explore and discover!
<br/>

# Installation

Brazilian Delight requires a few dependency mods to work properly, also keep in mind that it needs to be all in the same Minecraft version and modloader. Here's how to install them:

1. Download and install the [Farmer's Delight](https://modrinth.com/mod/farmers-delight) mod.
2. Download and install the [Deltabox Lib](https://modrinth.com/mod/deltaboxlib) mod.

If you are using Forge:
- Download and install the [Kotlin For Forge](https://modrinth.com/mod/kotlin-for-forge) mod.

Or if you are using Fabric:
- Download and install the [Fabric Language Kotlin](https://modrinth.com/mod/fabric-language-kotlin) mod.

Put all the downloaded mods in the `mods` folder inside your `.minecraft` directory.
